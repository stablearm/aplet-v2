# Page Rules

Every page must answer one business question.

Every page must expose one primary action.

Every page must be centered around one domain entity.

Pages must never duplicate functionality.

If two pages solve the same problem,

merge them.

---

# UX Rules

A page is successful when a user immediately understands:

Where am I?

What can I do?

What should I do next?

Nothing else matters before these three questions are answered.
